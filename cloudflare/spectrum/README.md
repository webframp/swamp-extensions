# @webframp/cloudflare/spectrum

Cloudflare Spectrum — TCP/UDP proxying for non-HTTP applications

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/spectrum

swamp model create @webframp/cloudflare/spectrum cf-spectrum \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg zoneId=YOUR_ZONE_ID
```
