# @webframp/cloudflare/api-shield

Cloudflare API Shield — schema validation, endpoint discovery, sequence rules

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/api-shield

swamp model create @webframp/cloudflare/api-shield cf-api-shield \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg zoneId=YOUR_ZONE_ID
```
