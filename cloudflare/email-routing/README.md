# @webframp/cloudflare/email-routing

Cloudflare Email Routing — rules, addresses, catch-all, DNS setup

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/email-routing

swamp model create @webframp/cloudflare/email-routing cf-email-routing \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg zoneId=YOUR_ZONE_ID
```
