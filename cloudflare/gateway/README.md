# @webframp/cloudflare/gateway

Cloudflare Gateway — DNS/HTTP policies, locations, proxy endpoints

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/gateway

swamp model create @webframp/cloudflare/gateway cf-gateway \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
