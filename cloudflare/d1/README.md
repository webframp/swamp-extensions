# @webframp/cloudflare/d1

Cloudflare D1 serverless SQL databases — databases, queries

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/d1

swamp model create @webframp/cloudflare/d1 cf-d1 \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
