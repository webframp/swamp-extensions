# @webframp/cloudflare/ssl-certificates

Cloudflare SSL/TLS — certificate packs, custom certificates, certificate
authority

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/ssl-certificates

swamp model create @webframp/cloudflare/ssl-certificates cf-ssl-certificates \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg zoneId=YOUR_ZONE_ID
```
