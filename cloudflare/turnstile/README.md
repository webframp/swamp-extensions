# @webframp/cloudflare/turnstile

Cloudflare Turnstile — CAPTCHA-free challenges, site widgets

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/turnstile

swamp model create @webframp/cloudflare/turnstile cf-turnstile \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```

## Methods

| Method              | Description                          |
| ------------------- | ------------------------------------ |
| `list`              | List Turnstile Widgets               |
| `get_create`        | Create a Turnstile Widget            |
| `get_get`           | Turnstile Widget Details             |
| `get_update`        | Update a Turnstile Widget            |
| `get_delete`        | Delete a Turnstile Widget            |
| `get_rotate_secret` | Rotate Secret for a Turnstile Widget |
