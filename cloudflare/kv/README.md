# @webframp/cloudflare/kv

Cloudflare Workers KV — namespaces, keys, values, bulk operations

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/kv

swamp model create @webframp/cloudflare/kv cf-kv \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
