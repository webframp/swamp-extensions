# @webframp/cloudflare/images

Cloudflare Images — upload, transform, deliver, and manage image pipelines

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/images

swamp model create @webframp/cloudflare/images cf-images \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
