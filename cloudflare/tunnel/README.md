# @webframp/cloudflare/tunnel

specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/tunnel

swamp model create @webframp/cloudflare/tunnel cf-tunnel \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
