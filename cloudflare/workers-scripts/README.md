# @webframp/cloudflare/workers-scripts

Cloudflare Workers Scripts — upload, deploy, bindings, routes, cron triggers

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/workers-scripts

swamp model create @webframp/cloudflare/workers-scripts cf-workers-scripts \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
