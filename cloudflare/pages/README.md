# @webframp/cloudflare/pages

Cloudflare Pages — projects, deployments, domains, build configs

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/pages

swamp model create @webframp/cloudflare/pages cf-pages \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
