# @webframp/cloudflare/r2

Cloudflare R2 object storage — buckets, objects, multipart uploads,
notifications

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/r2

swamp model create @webframp/cloudflare/r2 cf-r2 \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
