# @webframp/cloudflare/access

Cloudflare Access (Zero Trust) — applications, policies, identity providers,
certificates

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/access

swamp model create @webframp/cloudflare/access cf-access \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
