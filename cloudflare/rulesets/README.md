# @webframp/cloudflare/rulesets

Cloudflare Rulesets — WAF custom rules, transform rules, managed rulesets

Auto-generated by `scripts/cloudflare-codegen` from the Cloudflare OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/cloudflare/rulesets

swamp model create @webframp/cloudflare/rulesets cf-rulesets \
  --global-arg apiToken=CLOUDFLARE_API_TOKEN \
  --global-arg accountId=YOUR_ACCOUNT_ID
```
