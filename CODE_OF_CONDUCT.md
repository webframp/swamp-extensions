# Code of Conduct

Be respectful. Assume good faith. Keep disagreements technical and specific —
critique the code or the idea, not the person.

Harassment, personal attacks, and discriminatory language aren't tolerated in
issues, PRs, or any other project space. The maintainer will remove comments,
close threads, or block participants who violate this, at their discretion.

Report a violation by contacting the maintainer directly through GitHub.

This is intentionally short — the goal is a workable, welcoming project, not
a legal document.
