# Support

These are free, open source, best-effort extensions — support is
community-driven, not commercial.

## Getting help

- **Something's broken?** Open a [GitHub
  issue](https://github.com/webframp/swamp-extensions/issues) with the
  extension name, method, and what happened.
- **Have a question or an idea?** An issue works for this too.
- **Found a security issue?** Don't open a public issue — see
  [SECURITY.md](SECURITY.md).

## What this doesn't include

There's no SLA, no paid support tier, and no guarantee any given issue gets
fixed on any particular timeline. See [Support and
Warranty](README.md#support-and-warranty) in the README for the full picture.

A clear issue — ideally with a reproduction or a suggested fix included in the
description — is the fastest path to a resolution. Pull requests are only
occasionally accepted; see [CONTRIBUTING.md](CONTRIBUTING.md) for this repo's
contribution policy.
