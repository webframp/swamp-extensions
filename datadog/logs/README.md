# @webframp/datadog/logs

Datadog Logs — log search, aggregation, and analytics

Auto-generated by `scripts/datadog-codegen` from the Datadog v2 REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/datadog/logs

swamp model create @webframp/datadog/logs dd-logs \
  --global-arg apiKey=DD_API_KEY \
  --global-arg appKey=DD_APP_KEY \
  --global-arg site=us1
```

## Methods

| Method           | Description        |
| ---------------- | ------------------ |
| `submit_log`     | Send logs          |
| `aggregate_logs` | Aggregate events   |
| `list_logs_get`  | Search logs (GET)  |
| `list_logs`      | Search logs (POST) |
