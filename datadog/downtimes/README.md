# @webframp/datadog/downtimes

Datadog Downtimes — scheduled downtime management for monitors

Auto-generated by `scripts/datadog-codegen` from the Datadog v2 REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/datadog/downtimes

swamp model create @webframp/datadog/downtimes dd-downtimes \
  --global-arg apiKey=DD_API_KEY \
  --global-arg appKey=DD_APP_KEY \
  --global-arg site=us1
```

## Methods

| Method                   | Description                        |
| ------------------------ | ---------------------------------- |
| `list_downtimes`         | Get all downtimes                  |
| `create_downtime`        | Schedule a downtime                |
| `get_downtime`           | Get a downtime                     |
| `update_downtime`        | Update a downtime                  |
| `cancel_downtime`        | Cancel a downtime                  |
| `list_monitor_downtimes` | Get active downtimes for a monitor |
