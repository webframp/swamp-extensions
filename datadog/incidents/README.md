# @webframp/datadog/incidents

Datadog Incidents — incident lifecycle, timelines, teams, and attachments

Auto-generated by `scripts/datadog-codegen` from the Datadog v2 REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/datadog/incidents

swamp model create @webframp/datadog/incidents dd-incidents \
  --global-arg apiKey=DD_API_KEY \
  --global-arg appKey=DD_APP_KEY \
  --global-arg site=us1
```

## Methods

| Method                   | Description                |
| ------------------------ | -------------------------- |
| `list_incident_impacts`  | List an incident's impacts |
| `create_incident_impact` | Create an incident impact  |
| `delete_incident_impact` | Delete an incident impact  |
