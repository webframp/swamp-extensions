# @webframp/datadog/metrics

Datadog Metrics — metric queries, submissions, tag configurations, and metadata

Auto-generated by `scripts/datadog-codegen` from the Datadog v2 REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/datadog/metrics

swamp model create @webframp/datadog/metrics dd-metrics \
  --global-arg apiKey=DD_API_KEY \
  --global-arg appKey=DD_APP_KEY \
  --global-arg site=us1
```

## Methods

| Method                               | Description                                    |
| ------------------------------------ | ---------------------------------------------- |
| `list_tag_configurations`            | Get a list of metrics                          |
| `list_active_metric_configurations`  | List active tags and aggregations              |
| `list_tags_by_metric_name`           | List tags by metric name                       |
| `list_metric_assets`                 | Related Assets to a Metric                     |
| `estimate_metrics_output_series`     | Tag Configuration Cardinality Estimator        |
| `get_metric_tag_cardinality_details` | Get tag key cardinality details                |
| `list_tag_configuration_by_name`     | List tag configuration by name                 |
| `create_tag_configuration`           | Create a tag configuration                     |
| `update_tag_configuration`           | Update a tag configuration                     |
| `delete_tag_configuration`           | Delete a tag configuration                     |
| `list_volumes_by_metric_name`        | List distinct metric volumes by metric name    |
| `query_scalar_data`                  | Query scalar data across multiple products     |
| `query_timeseries_data`              | Query timeseries data across multiple products |
| `submit_metrics`                     | Submit metrics                                 |
