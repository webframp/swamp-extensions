# @webframp/datadog/synthetics

Datadog Synthetics — synthetic monitoring tests, results, and locations

Auto-generated by `scripts/datadog-codegen` from the Datadog v2 REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/datadog/synthetics

swamp model create @webframp/datadog/synthetics dd-synthetics \
  --global-arg apiKey=DD_API_KEY \
  --global-arg appKey=DD_APP_KEY \
  --global-arg site=us1
```
