# @webframp/snyk/inventory

Snyk Inventory — asset discovery for packages, containers, repos, and cloud
resources

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/inventory

swamp model create @webframp/snyk/inventory snyk-inventory \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```
