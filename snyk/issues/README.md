# @webframp/snyk/issues

Snyk Issues — vulnerability issues across projects and groups

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/issues

swamp model create @webframp/snyk/issues snyk-issues \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                        | Description            |
| ----------------------------- | ---------------------- |
| `list_group_issues`           | Get issues by group ID |
| `get_group_issue_by_issue_id` | Get an issue           |
| `list_org_issues`             | Get issues by org ID   |
| `get_org_issue_by_issue_id`   | Get an issue           |

## Development

```bash
cd ../../snyk/issues
deno task check
deno task lint
deno task fmt
deno task test
```
