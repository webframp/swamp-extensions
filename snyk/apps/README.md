# @webframp/snyk/apps

Snyk Apps — OAuth application management, bots, installations

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/apps

swamp model create @webframp/snyk/apps snyk-apps \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                              | Description                                                                                                |
| ----------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| `get_app_installs_for_group`        | Get a list of Snyk Apps installed for a Group                                                              |
| `create_group_app_install`          | Install a Snyk App for a Group                                                                             |
| `delete_group_app_install_by_id`    | Revoke app authorization for a Snyk group with install ID                                                  |
| `update_group_app_install_secret`   | Manage client secret for non-interactive Snyk App installations                                            |
| `get_org_apps`                      | Get a list of Snyk Apps created by an Organization                                                         |
| `create_org_app`                    | Create a new Snyk App for an organization                                                                  |
| `get_app_by_id`                     | Get a Snyk App by app ID                                                                                   |
| `update_app_creation_by_id`         | Update app creation attributes such as name, redirect URIs, and access token time to live using the App ID |
| `delete_app_by_id`                  | Delete a Snyk App by app ID                                                                                |
| `create_manage_app_creation_secret` | Manage client secret for a Snyk App                                                                        |
| `get_app_installs_for_org`          | Get a list of Snyk Apps installed for an Organization                                                      |
| `create_org_app_install`            | Install a Snyk App for an Organization                                                                     |
| `delete_app_org_install_by_id`      | Revoke app authorization for a Snyk organization with install ID                                           |
| `update_org_app_install_secret`     | Manage client secret for non-interactive Snyk App installations                                            |
