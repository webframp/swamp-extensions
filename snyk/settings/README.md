# @webframp/snyk/settings

Snyk Settings — organization and group setting management

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/settings

swamp model create @webframp/snyk/settings snyk-settings \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```
