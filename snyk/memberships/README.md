# @webframp/snyk/memberships

Snyk Memberships — group and org member management

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/memberships

swamp model create @webframp/snyk/memberships snyk-memberships \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                         | Description                                    |
| ------------------------------ | ---------------------------------------------- |
| `list_group_memberships`       | Get all memberships of the group               |
| `create_group_membership`      | Create a group membership for a user with role |
| `update_group_user_membership` | Update a role from a group membership          |
| `delete_group_membership`      | Delete a membership from a group               |
| `list_org_memberships`         | Get all memberships of the org                 |
| `create_org_membership`        | Create a org membership for a user with role   |
| `update_org_membership`        | Update a org membership for a user with role   |
| `delete_org_membership`        | Remove user's org membership                   |
