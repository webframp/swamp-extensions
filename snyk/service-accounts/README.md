# @webframp/snyk/service-accounts

Snyk Service Accounts — automated access management for CI/CD

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/service-accounts

swamp model create @webframp/snyk/service-accounts snyk-service-accounts \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                              | Description                                             |
| ----------------------------------- | ------------------------------------------------------- |
| `get_many_group_service_account`    | Get a list of group service accounts.                   |
| `create_group_service_account`      | Create a service account for a group.                   |
| `get_one_group_service_account`     | Get a group service account.                            |
| `update_group_service_account`      | Update a group service account.                         |
| `delete_one_group_service_account`  | Delete a group service account.                         |
| `update_service_account_secret`     | Manage a group service account's client secret.         |
| `get_many_org_service_accounts`     | Get a list of organization service accounts.            |
| `create_org_service_account`        | Create a service account for an organization.           |
| `get_one_org_service_account`       | Get an organization service account.                    |
| `update_org_service_account`        | Update an organization service account.                 |
| `delete_service_account`            | Delete a service account in an organization.            |
| `update_org_service_account_secret` | Manage an organization service account's client secret. |
