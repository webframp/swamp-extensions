# @webframp/snyk/collections

Snyk Collections — project collection groupings and management

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/collections

swamp model create @webframp/snyk/collections snyk-collections \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                            | Description                                |
| --------------------------------- | ------------------------------------------ |
| `get_collections`                 | Get collections                            |
| `create_collection`               | Create a collection                        |
| `get_collection`                  | Get a collection                           |
| `update_collection`               | Edit a collection                          |
| `delete_collection`               | Delete a collection                        |
| `get_projects_of_collection`      | Get projects from the specified collection |
| `update_collection_with_projects` | Add projects to a collection               |
| `delete_projects_collection`      | Remove projects from a collection          |
