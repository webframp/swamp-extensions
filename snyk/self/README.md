# @webframp/snyk/self

Snyk Self — current user context, org listing, and app management

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/self

swamp model create @webframp/snyk/self snyk-self \
  --global-arg apiToken=SNYK_API_TOKEN
```

## Methods

| Method                             | Description                                         |
| ---------------------------------- | --------------------------------------------------- |
| `get_self`                         | My User Details                                     |
| `get_access_requests`              | Get access requests (Early Access)                  |
| `get_user_installed_apps`          | Get a list of Snyk Apps that can act on your behalf |
| `get_app_installs_for_user`        | Get a list of Snyk Apps installed for a user        |
| `delete_user_app_install_by_id`    | Revoke a Snyk App by install ID                     |
| `delete_revoke_user_installed_app` | Revoke a Snyk App by app ID                         |
| `get_user_app_sessions`            | Get a list of active OAuth sessions by app ID       |
| `delete_revoke_user_app_session`   | Revoke the Snyk App session of an active user       |
| `list_personal_access_token`       | List personal access tokens                         |
| `delete_personal_access_token`     | Deletes a personal access token                     |
