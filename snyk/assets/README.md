# @webframp/snyk/assets

Snyk Assets — asset discovery and classification across the group

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/assets

swamp model create @webframp/snyk/assets snyk-assets \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg groupId=YOUR_GROUP_ID
```

## Methods

| Method                | Description                                        |
| --------------------- | -------------------------------------------------- |
| `list_assets`         | List Assets with filters (Early Access)            |
| `get_asset`           | Get an Asset by its ID (Early Access)              |
| `list_related_assets` | List related assets with pagination (Early Access) |
| `list_asset_projects` | List asset projects with pagination (Early Access) |
