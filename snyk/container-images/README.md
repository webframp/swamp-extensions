# @webframp/snyk/container-images

Snyk Container Images — container image scanning and vulnerability data

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/container-images

swamp model create @webframp/snyk/container-images snyk-container-images \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                   | Description                                                     |
| ------------------------ | --------------------------------------------------------------- |
| `list_container_image`   | List instances of container image                               |
| `get_container_image`    | Get instance of container image                                 |
| `list_image_target_refs` | List instances of image target references for a container image |
