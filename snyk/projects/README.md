# @webframp/snyk/projects

Snyk Projects — project listing, attributes, relationships, and target
management

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/projects

swamp model create @webframp/snyk/projects snyk-projects \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method               | Description                                         |
| -------------------- | --------------------------------------------------- |
| `list_org_projects`  | List all Projects for an Org with the given Org ID. |
| `get_org_project`    | Get project by project ID.                          |
| `update_org_project` | Updates project by project ID.                      |
| `delete_org_project` | Delete project by project ID.                       |
| `get_sbom`           | Get a project’s SBOM document                       |
