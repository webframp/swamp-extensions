# @webframp/snyk/sbom

Snyk SBOM — software bill of materials testing and analysis

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/sbom

swamp model create @webframp/snyk/sbom snyk-sbom \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                 | Description                                 |
| ---------------------- | ------------------------------------------- |
| `create_sbom_test_run` | Create an SBOM test run (Early Access)      |
| `get_sbom_test_status` | Gets an SBOM test run status (Early Access) |
| `get_sbom_test_result` | Gets an SBOM test run result (Early Access) |

## Development

cd ../../snyk/sbom
