# @webframp/snyk/tenants

Snyk Tenants — tenant and organization lifecycle management

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/tenants

swamp model create @webframp/snyk/tenants snyk-tenants \
  --global-arg apiToken=SNYK_API_TOKEN
```
