# @webframp/snyk/slack

Snyk Slack Integration — Slack app configuration and channel management

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/slack

swamp model create @webframp/snyk/slack snyk-slack \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                                               | Description                                               |
| ---------------------------------------------------- | --------------------------------------------------------- |
| `get_slack_default_notification_settings`            | Get Slack integration default notification settings.      |
| `create_slack_default_notification_settings`         | Create new Slack notification default settings.           |
| `delete_slack_default_notification_settings`         | Remove the given Slack App integration                    |
| `get_slack_project_notification_settings_collection` | Slack notification settings overrides for projects        |
| `create_slack_project_notification_settings`         | Create a new Slack settings override for a given project. |
| `update_slack_project_notification_settings`         | Update Slack notification settings for a project.         |
| `delete_slack_project_notification_settings`         | Remove Slack settings override for a project.             |
| `list_channels`                                      | Get a list of Slack channels                              |
| `get_channel_name_by_id`                             | Get Slack Channel name by Slack Channel ID.               |
