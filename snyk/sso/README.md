# @webframp/snyk/sso

Snyk SSO — single sign-on connection management for groups

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/sso

swamp model create @webframp/snyk/sso snyk-sso \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg groupId=YOUR_GROUP_ID
```

## Methods

| Method                            | Description                                               |
| --------------------------------- | --------------------------------------------------------- |
| `list_group_sso_connections`      | Get all SSO connections for a group (Early Access)        |
| `list_group_sso_connection_users` | Get all users using a given SSO connection (Early Access) |
| `delete_user`                     | Delete a user from a Group SSO connection (Early Access)  |
