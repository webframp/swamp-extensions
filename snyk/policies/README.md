# @webframp/snyk/policies

Snyk Policies — security policy management and rule configuration

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/policies

swamp model create @webframp/snyk/policies snyk-policies \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg orgId=YOUR_ORG_ID
```

## Methods

| Method                  | Description                           |
| ----------------------- | ------------------------------------- |
| `get_org_policies`      | Get org-level policies                |
| `create_org_policy`     | Create a new org-level policy         |
| `get_org_policy`        | Get an org-level policy               |
| `update_org_policy`     | Update an org-level policy            |
| `delete_org_policy`     | Delete an org-level policy            |
| `get_org_policy_events` | List org policy events (Early Access) |
