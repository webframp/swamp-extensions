# @webframp/snyk/groups

Snyk Groups — group management, orgs, members, and audit

Auto-generated by `scripts/snyk-codegen` from the Snyk REST API OpenAPI
specification.

## Usage

```bash
swamp extension pull @webframp/snyk/groups

swamp model create @webframp/snyk/groups snyk-groups \
  --global-arg apiToken=SNYK_API_TOKEN \
  --global-arg groupId=YOUR_GROUP_ID
```

## Methods

| Method                            | Description                                                       |
| --------------------------------- | ----------------------------------------------------------------- |
| `list_groups`                     | Get all groups (Early Access)                                     |
| `get_group`                       | Get Group (Early Access)                                          |
| `list_group_audit_logs`           | Search Group audit logs.                                          |
| `create_group_export`             | Start an export                                                   |
| `get_group_export`                | Get export results                                                |
| `get_group_export_job_status`     | Get export status                                                 |
| `list_group_user_org_memberships` | Get list of org memberships of a group user                       |
| `list_orgs_in_group`              | List all organizations in group                                   |
| `list_group_policies`             | Get group level policies (Early Access)                           |
| `create_group_policy`             | Create a new group level policy (Early Access)                    |
| `update_group_policy`             | Update a group-level policy (Early Access)                        |
| `delete_group_policy`             | Delete an group-level policy (Early Access)                       |
| `list_assignments`                | List all assignments for a group                                  |
| `create_assignments`              | Create bulk assignments for rule extensions                       |
| `delete_assignments`              | Delete assignments for rule extensions                            |
| `list_secrets_rule_extensions`    | Get a list of all secrets rule extensions for the requested Group |
| `create_secrets_rule_extension`   | Create a secrets rule extension                                   |
| `get_secrets_rule_extension`      | Get a secrets rule extension by rule extension ID                 |
| `update_secrets_rule_extension`   | Update a secrets rule extension by rule extension ID              |
| `delete_secrets_rule_extension`   | Delete a secrets rule extension by rule extension ID              |
| `update_user`                     | Update a user's role in a group (Early Access)                    |
